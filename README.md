# JARL Public Log SCP Generator

JARLコンテストのパブリックログを自動ダウンロードし、N1MM Logger+ や DXLog 等で利用できる Super Check Partial（SCP）ファイルを生成する Python スクリプトです。

外部ライブラリは不要で、Python 3.12 標準ライブラリのみで動作します。

---

## 動作環境

- Python 3.12 以降
- インターネット接続

---

## インストール

任意のフォルダに以下のファイルを保存してください。

```
spc_generator_for_JA.py
```

---

## 実行方法

コマンドプロンプトまたはターミナルを開き、スクリプトを保存したフォルダで以下を実行してください。

```cmd
python spc_generator_for_JA.py
```

---

## 起動時メニュー

起動すると以下のメニューが表示されます。

```
コンテストを選択してください
ALL JA      : j
6m AND DOWN : 6
Field Day   : f
全市全郡     : a
その他       : o
```

生成したいコンテストに対応するキーを入力してください。

## 対応コンテスト

| キー | コンテスト |
| --- | --- |
| j | ALL JA |
| 6 | 6m AND DOWN |
| f | Field Day |
| a | 全市全郡（ACAG） |
| o | その他 |

---

## パブリックログの取得

起動時メニューで選択したコンテストに関わらず、以下の4コンテスト全てのパブリックログを自動ダウンロードします。

- ALL JA
- 6m AND DOWN
- Field Day
- 全市全郡

**対象年度：** スクリプト実行時の年から過去5年分（ただし2025年以降のみ）

ダウンロードしたZIPファイルはキャッシュフォルダに保存され、次回以降の実行では再ダウンロードされません。

### キャッシュフォルダ

```
.jarl_publiclog_cache/
```

スクリプトと同じフォルダに自動作成されます。

## SCPファイルのフォーマット

### 通常コンテスト（ALL JA / 6m AND DOWN / Field Day）

コールサインとコンテストナンバーをセットで出力します。

```
JA1ZLO     10H
JJ1AHS     12M
JR1ZTT     106H
```

コンテストナンバーは都道府県ナンバー（2〜3桁の数字＋パワー記号）を使用します。

同一コールサインについて複数のコンテストやバンドにまたがる記録がある場合、選択したコンテスト・バンドの記録を優先して採用します。同年度内に複数のナンバーが存在する場合は全て出力します。

### 全市全郡（ACAG）および SH バンド

コンテストナンバーとして市郡区ナンバー（4桁以上の数字＋パワー記号）を使用します。

市郡区ナンバーが存在する場合：

```
JA1AAA     100110H
JJ1BBB     1420M
```

市郡区ナンバーが存在しない場合：

```
JA1AAA
JJ1BBB
```

コールサインのみ出力されます。

### その他モード

全コンテストのパブリックログから都道府県ナンバー（2〜3桁）を持つ記録を集計し、パワー記号を除いたナンバーを出力します。

パブリックログ：

```
JA1AAA 10H
JA1BBB 10M
JA1CCC 106L
```

生成結果：

```
JA1AAA     10
JA1BBB     10
JA1CCC     106
```

## エラー処理

以下の状況が発生しても処理を中断せず継続します。

- 指定年度のパブリックログが未公開または存在しない
- ダウンロード失敗（HTTP エラー・タイムアウト等）
- ZIP ファイルの破損
- 文字コードの判定失敗

いずれの場合も警告メッセージのみ表示されます。

---

## 文字コード

ZIP 内のログファイルは以下の順で文字コードを自動判定します。

1. UTF-8
2. Shift_JIS
3. CP932
